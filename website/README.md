# 不想排座位 - 宣传官网

这是“不想排座位”项目的官方宣传网页，使用 React + Vite 构建，设计风格简洁现代。

## 开发指南

1. 进入目录：
   ```bash
   cd website
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 构建生产版本：
   ```bash
   npm run build
   ```

## Vercel 一键部署

本项目已预配置为 Vercel 友好型项目。

1. 将本仓库推送到 GitHub。
2. 登录 [Vercel](https://vercel.com)。
3. 点击 "Add New Project" 并导入您的仓库。
4. **关键步骤**：在 "Root Directory"（根目录）设置中，点击 "Edit" 并选择 `website` 文件夹。
5. 框架预设（Framework Preset）会自动识别为 `Vite`。
6. 点击 "Deploy"。

部署完成后，你将获得一个免费的 HTTPS 域名用于访问官网。
